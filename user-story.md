# User Story Template

## Format
**As a** [type of user],
**I want** [an action or feature],
**So that** [benefit or value].

---

## Examples

### Example 1: Account Creation
**As a** customer,
**I want** to create a new account,
**So that** I can manage my profile and transactions.

### Example 2: Read Account
**As a** customer,
**I want** to view my account details,
**So that** I can check my balance and information.

### Example 3: Update Account
**As a** customer,
**I want** to update my account information,
**So that** my details are always current.

### Example 4: Delete Account
**As a** customer,
**I want** to delete my account,
**So that** I can close my profile when no longer needed.

### Example 5: List Accounts
**As an** admin,
**I want** to list all customer accounts,
**So that** I can manage and monitor all users.
